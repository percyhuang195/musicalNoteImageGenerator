- 一個以 Kotlin 語言編寫，可以用於產生五線譜元件的插件

## 功能介紹
- 透過滑桿(Slider)控制音符的音高
- 可以新增、刪除音符
- 可以在音符側邊產生升降記號

## 安裝說明
- 請下載 Release 區域內任一版本的 APK 檔案，將提供可在 Android 裝置運行的範例 APP
- 插件功能仍在開發中
